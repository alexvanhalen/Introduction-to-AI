## 題目六：簡易文字加密解密器 🔐

### 📝 題目描述
實作凱薩密碼(Caesar Cipher)，讓使用者可以加密和解密訊息。

### 🎯 學習目標
- 字串處理進階技巧
- ASCII 編碼與字元轉換
- 加密演算法理解
- 函數設計與模組化

### ✅ 基本要求
1. 實作加密功能(輸入明文和位移量，輸出密文)
2. 實作解密功能(輸入密文和位移量，輸出明文)
3. 至少支援英文字母(大小寫)
4. 保持非字母字元不變(如標點符號、空格)
5. 提供使用者友善的選單介面

### 🚀 進階挑戰
- 實作暴力破解功能(嘗試所有 26 種位移)
- 支援中文字加密(使用 Unicode)
- 實作其他加密法(如 Vigenère cipher)
- 加入密碼強度分析
- 支援檔案加密/解密

### 💡 提示
```python
# 凱薩密碼範例
def caesar_encrypt(text, shift):
    result = ""
    for char in text:
        if char.isalpha():
            # 處理大小寫
            base = ord('A') if char.isupper() else ord('a')
            result += chr((ord(char) - base + shift) % 26 + base)
        else:
            result += char
    return result
```

5. 提供開始、暫停功能

### 🚀 進階挑戰
- 顯示進度條(使用 tqdm 套件)
- 時間到時播放提醒音效或顯示通知
- 製作今日專注時間統計圖表
- 可自訂工作/休息時長
- 整合待辦清單功能
- 每完成 4 個番茄鐘後，給予較長的休息時間(15-20分鐘)

### 💡 提示
```python
import time
from datetime import datetime, timedelta

# 簡易倒數計時範例
def countdown(minutes):
    total_seconds = minutes * 60
    while total_seconds > 0:
        mins, secs = divmod(total_seconds, 60)
        timer = f'{mins:02d}:{secs:02d}'
        print(timer, end='\r')
        time.sleep(1)
        total_seconds -= 1
```
